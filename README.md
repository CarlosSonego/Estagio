# Estagio
Este trabalho tem como objetivo a realização de um site para a realização do estagio obrigatorio da UNESPAR, sendo realizado em conjunto com o TCC.<br/>
Este site irá mostrar os resultados obtidos na pesquisa do TCC com o seguinte tema: Utilização de IA generativa para 
explicar predições no futebol Brasileiro.<br/>
Disponivel em https://carlossonego.github.io/Estagio/
